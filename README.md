# 🚀 FastAPI + Vue 聊天室
一个基于现代 Web 技术的实时聊天应用，后端使用 FastAPI 提供 RESTful API 和 socket.IO 支持，前端使用 Vue 3 构建响应式界面。
### 后端 (FastAPI)
- JWT 用户认证
- socket.IO 实时消息推送
- 消息历史记录存储
- RESTful API 文档 (自动生成 Swagger UI)
- PostgreSQL 数据库集成
## 🛠️ 技术栈

| 组件         | 技术选型                             |
|--------------|------------------------------------|
| **后端框架**  | FastAPI                           |
| **前端框架**  | Vue 3                |
| **数据库**    | PostgreSQL         |
| **实时通信**  | socket.IO                         |
| **状态管理**  | Pinia                             |
| **样式框架**  | Tailwind CSS                      |
